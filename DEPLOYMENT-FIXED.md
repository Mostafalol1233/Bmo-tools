## Quick Deploy Steps

### Option 1: Vercel Dashboard
1. Go to [vercel.com](https://vercel.com)
2. Click "New Project"
3. Import from GitHub
4. Framework: **Other**
5. Build Command: `npm run build`
6. Output Directory: `dist`
7. Deploy

### Option 2: CLI Deploy
```bash
npm install -g vercel
vercel login
vercel --prod
```
